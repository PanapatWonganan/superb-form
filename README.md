## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the survey form.

## Customization

You can customize the survey questions by editing the `questions` array in `components/SurveyForm.tsx`.
